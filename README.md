需求描述：  
存在四个对象：
我方飞机、地方飞机、我方子弹、敌方子弹

功能：  
背景音乐的添加  
我方飞机可以移动（根据按键来控制）  
地方飞机也可以移动（随机的主动移动）  
双方飞机都可以发射子弹

步骤：  
1、创建一个窗口  
2、创建一个我方飞机，根据方向键左右移动  
3、给我方飞机添加发射子弹的功能（按下空格键去发射）  
4、创建一个敌方飞机  
5、敌方飞机可以自由移动  
6、敌方飞机可以自动发射子弹
